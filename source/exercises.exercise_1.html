exercises.exercise_1.exercise_1.get_duplicates(lst: list, comparator=<function <lambda>>) → list¶

This function returns a list of duplicate elements in a given list. It is compatible with hashable or non-hashable elements in the input, as long as the object type does have the comparator definition.

It is not as efficient as the only-hashable objects solution (which is displayed in module exercise_1_hashable_objects.py). In here, any new item is added to the appeared list. If it is already present in the appeared list, we should see if it is present in the result list. If not, the element is added to the result list.

Parameters:

  • lst (list) – The input list (can contain any type of objects).

  • comparator (function) – The equality of duplicates operator (could be ‘==’, or ‘is’, or any other comparator)

Returns:

A list containing duplicate elements (“duplicate” in this case refers to the comparator defined equality).

Return type:

list

exercises.exercise_1.exercise_1_hashable_objects.get_duplicates(lst: list) → list¶

This function returns a list of duplicate elements in a given list. It is compatible only with hashable elements in the input.

This solution is more efficient than the previous one, because it uses a dictionary, whose keys (objects) are stored inside a hashmap. Thus, key lookup is on average O(1). :param lst: the input list of hashable elements :type lst: list :return: the duplicates list :rtype: list
